BBSearch

BBSearch is a jQuery plugin that takes an input-field and transform it into a remote-auto-search element based on Backbone

  • BBSearch's code
  • BBSearch's demo page
  • BBSearch's README

Demo page: play with the examples

Use

$(<selector>).bbsearch({
  url:              <* url of the search API, the search query will be sent into a 'q' param>,
  itemTemplate:     <* Underscore-style template for the result elements>,
  resultsElement:   <* The results container element, an 'ul' element will be appended to it>,
  loadingElement:   <The loading element will be shown and hidden while fetching>
  onClick:          <onclick handler>,
  onRequest:        <start request handler>,
  onResponse:       <end of request handler>,
  parse:            <overriding the default Backbone.Collection.parse()>,
});

* required

Example 1: Twitter Search API (simple)

This remote-auto-search element is connected to Twitter search API. Click any text on it and wait for results.

Example 2: Twitter Search API (styled)

This remote-auto-search element is connected to Twitter search API. Write any text on it and wait for results. The results are clickable and the twitt will be setted at the input field.
